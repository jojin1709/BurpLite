🎯 Intercept

Real request interception (hold ALL requests before they reach the server, Forward / Drop / Race) runs in the Side Panel — click the HackTools++ toolbar icon to open it. It uses chrome.debugger, which can't run while DevTools is open on the same tab, so it lives outside DevTools.

    Empty = capture all

        Quick Start
        1. Capture any request from the target site.
        2. Click request in Captured Requests list.
        3. Edit method/headers/body and click Send.
